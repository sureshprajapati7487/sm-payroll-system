# SM Payroll System

A comprehensive Payroll & Workforce Management System built for Indian Manufacturing/Service sectors.

## 🚀 Features

### 👥 Employee Management
- Complete Profile (Personal, Bank, Salary Details).
- Role-Based Access (Super Admin, Admin, Account, Manager, Employee).
- **Security**: Employees can only see their own data.

### ⏱️ Smart Attendance
- **Face Scan Simulation**: Web UI for checking in/out.
- **Auto-Late Marking**: Linked to Shift timings (e.g., 9:00 AM + 15m Grace).
- **Manual Override**: Admins can fix attendance errors.

### 🏭 Production Tracking
- **Piece Rate System**: Track work by Item/Qty/Rate.
- **Approval Flow**: Managers Submit -> Admins Approve.
- **Real-time Stats**: Daily Earnings calculation.

### 💸 Loan & Advance
- **Issue Loans**: Cash Advance, PF Loan, Fine, etc.
- **EMI Management**: Auto-deduction from Salary.
- **Flexible Repayment**: Admins can adjust EMI amounts.

### 💰 Payroll Engine
- **One-Click Generation**: Calculates Salary for the whole company.
- **Logic**:
  - `(Basic / Days) * PresentDays` + `Overtime * 1.5`
  - `Production Total Amount`
  - `(-) Loan Deductions`
- **Payslip**: Professional Printable PDF format.

### 📊 Executive Dashboard
- **Real-time KPIs**: Presence %, Production Value, Active Staff.
- **Charts**: Attendance Trends, Dept Performance.
- **Role-Views**: Customized views for Admin vs Manager vs Employee.

## 🛠️ Technology Stack
- **Frontend**: React (Vite) + TypeScript.
- **UI**: Tailwind CSS + Lucide Icons + Glassmorphism Design.
- **State Management**: Zustand.
- **Routing**: React Router DOM (Protected Routes).
- **Charts**: Recharts.

## 🏃‍♂️ How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```
    The output will be in the `dist` folder.
